# 3월 20일 교육 사전 설치 가이드

**수요일 미팅 전까지 완료해주세요.**
설치 중 막히는 부분 있으면 수요일에 같이 확인합니다.

---

## 전원 공통

### 1. VSCode 설치
https://code.visualstudio.com

### 2. Node.js 설치 (미설치 시)
https://nodejs.org (LTS 버전)

설치 확인:
```bash
node --version
npm --version
```

### 3. Claude Code 설치
```bash
npm install -g @anthropic-ai/claude-code
```

설치 확인:
```bash
claude --version
```

### 4. Claude Code 로그인
```bash
claude
```
→ 브라우저 열리면 Claude 계정으로 로그인

### 5. Playwright MCP 설치
VSCode settings.json에 추가:

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@playwright/mcp@latest"]
    }
  }
}
```

설치 확인: Claude Code에서 `/mcp` 입력 후 playwright 보이면 완료

---

## Max 계정 2명만 추가

### 6. Superpowers 플러그인 설치
Claude Code에서 실행:
```
/plugin marketplace add obra/superpowers-marketplace
```

설치 확인: Claude Code에서 `/help` 입력 후
`/superpowers:brainstorm` 등 커맨드 보이면 완료

터미널에서:
```bash
claude --dangerously-skip-permissions
```
→ Claude Code 실행되면 준비 완료
