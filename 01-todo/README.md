# 1.1.test-todo

바이브코딩 워크샵 시연용 Todo 앱.
Claude Code에 프롬프트 하나로 만든 결과물.

---

## 어떻게 만들었나

```bash
claude --dangerously-skip-permissions
```

**1차 프롬프트:**
```
HTML CSS JS 로컬스토리지 기반 Todo 앱 만들어줘
```

**2차 프롬프트 (버그 확인):**
```
완성된 앱의 모든 기능이 정상 동작하는지
확인하고 동작 안 하는 것 있으면 바로 수정해줘.
```

총 소요 시간: 약 4분 15초

---

## 실행 방법

```bash
# 브라우저로 바로 열기
open index.html
```

서버 불필요. `index.html` 더블클릭으로 실행.

---

## 기능

- 할 일 추가 / 완료 토글 / 삭제
- 필터 (All / Active / Done)
- 완료 항목 일괄 삭제
- 로컬스토리지 저장 (새로고침 후에도 유지)

---
