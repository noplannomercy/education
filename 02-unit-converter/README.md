# 1.2.test-unit-converter

바이브코딩 워크샵 시연용 단위환산기 웹앱.
PRD 작성 후 Superpowers로 구현한 결과물.

---

## 어떻게 만들었나

### 1단계 — PRD 작성

```
PRD 먼저 작성해줘.
앱 이름: 단위환산기
핵심 기능: 12개 카테고리 단위환산, 즐겨찾기, 히스토리
스택: HTML CSS JS 로컬스토리지
```

AI가 질문하면 답하면서 `docs/PRD.md` 완성.

### 2단계 — 구현 플랜 생성

Superpowers가 PRD를 읽고 태스크별 구현 플랜 자동 생성 → `docs/plans/2026-03-07-unit-converter.md`

### 3단계 — 태스크별 실행

Superpowers가 플랜의 태스크를 하나씩 실행. 태스크마다 서브에이전트가 독립 실행.

---

## 실행 방법

```bash
# 브라우저로 바로 열기
open index.html
```

서버 불필요. `index.html` 더블클릭으로 실행.

---

## 기능

- 12개 카테고리 단위환산 (길이/무게/온도/부피/속도/화폐 등)
- 양방향 입력 (From ⇄ To 실시간 환산)
- 즐겨찾기 저장/복원
- 변환 히스토리 최대 50건
- 다크모드 (시스템 설정 자동 감지 + 수동 토글)
- 반응형 (모바일/태블릿/데스크탑)
- LocalStorage 영속 저장
