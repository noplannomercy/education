# PRD: 단위환산기 웹앱 (Unit Converter)

**버전:** 1.0
**작성일:** 2026-03-07
**스택:** HTML · CSS · JavaScript (Vanilla) · LocalStorage

---

## 1. 개요

순수 프론트엔드 단일 HTML 파일로 구현하는 단위환산기 웹앱. 외부 서버나 빌드 도구 없이 브라우저에서 직접 실행되며, 사용자의 변환 기록과 즐겨찾기를 LocalStorage에 영속 저장한다.

---

## 2. 목표

- 12개 카테고리의 단위 환산을 빠르고 직관적으로 제공한다.
- 히스토리와 즐겨찾기로 반복 작업을 최소화한다.
- 파일 하나(`index.html`)만으로 배포·실행이 가능하다.

## 4. 지원 단위 카테고리 (12개)

| # | 카테고리 | 주요 단위 예시 |
|---|---------|--------------|
| 1 | 길이 | mm, cm, m, km, in, ft, yd, mi |
| 2 | 무게 | mg, g, kg, t, oz, lb |
| 3 | 온도 | °C, °F, K |
| 4 | 부피 | ml, L, m³, fl oz, pt, qt, gal |
| 5 | 넓이 | mm², cm², m², km², in², ft², ac |
| 6 | 속도 | m/s, km/h, mph, knot |
| 7 | 시간 | ms, s, min, h, day, week, month, year |
| 8 | 압력 | Pa, hPa, kPa, bar, atm, psi |
| 9 | 데이터 용량 | bit, B, KB, MB, GB, TB, PB |
| 10 | 에너지 | J, kJ, cal, kcal, Wh, kWh, BTU |

> **화폐:** 실시간 API 연동 없이 로컬 고정 환율 사용. UI에 기준 날짜를 표시한다.

## 5. 기능 요구사항

### 5.1 핵심 환산 기능

- **카테고리 선택** → 해당 카테고리의 단위 목록 표시
- **단위 쌍 선택** (From / To)
- **값 입력 시 실시간 환산** (onChange 이벤트)
- **양방향 입력**: From 입력 시 To 계산, To 수정 시 From 역산
- **단위 교체 버튼** (From ⇄ To 스왑)
- **결과 복사 버튼** (클립보드)

### 5.2 즐겨찾기

- 현재 카테고리 + 단위 쌍을 즐겨찾기에 저장/삭제 (토글)
- 즐겨찾기 클릭 시 해당 환산 화면으로 즉시 이동
- 저장 위치: `localStorage["uc_favorites"]`

### 5.3 변환 히스토리

- 환산 실행 시 자동 저장 (입력값이 유효한 숫자일 때)
- 최대 50건 유지 (초과 시 오래된 항목 자동 삭제)
- 타임스탬프 표시 (상대시간: "3분 전", "어제" 등)
- 히스토리 항목 클릭 시 해당 환산 복원
- 전체 히스토리 삭제 버튼
- 저장 위치: `localStorage["uc_history"]`

### 5.4 UI/UX

- 다크모드: 시스템 `prefers-color-scheme` 자동 감지 + 수동 토글
- 반응형: 모바일(< 768px) / 태블릿(768–1024px) / 데스크탑(> 1024px)
- 검색: 카테고리 또는 단위명 키워드 검색

## 6. UI 레이아웃

### 데스크탑 (> 768px) — 2컬럼 대시보드

```
┌─────────────────┬────────────────────────────────────────┐
│   사이드바       │   메인 패널                             │
│   (240px 고정)  │                                        │
│                 │   [카테고리명]          [즐겨찾기 토글]  │
│   [검색창]      │                                        │
│   ─────────     │   ┌─────────────┐  ⇄  ┌────────────┐ │
│   길이          │   │ FROM 단위   │     │ TO 단위    │ │
│   무게          │   │ [입력값]    │     │ [결과값]   │ │
│   온도          │   └─────────────┘     └────────────┘ │
│   ...           │   [복사]                               │
│   ─────────     │                                        │
│   즐겨찾기      │   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│   항목 목록     │   [즐겨찾기 탭] [히스토리 탭]           │
│                 │   항목 목록                             │
└─────────────────┴────────────────────────────────────────┘
```

### 모바일 (< 768px)

- 사이드바 숨김 → 상단 햄버거 메뉴로 드로어(Drawer) 열기
- 환산 패널 풀 너비
- 하단 탭바: [환산] [즐겨찾기] [히스토리]

## 8. 내부 아키텍처 (JS 모듈 구조)

단일 `<script>` 블록 내에서 네임스페이스로 분리:

| 객체 | 역할 |
|------|------|
| `UnitData` | 카테고리·단위·환산 계수 정의 (순수 데이터) |
| `Converter` | 환산 계산 로직 (`convert(category, from, to, value)`) |
| `Storage` | LocalStorage CRUD (히스토리, 즐겨찾기) |
| `UI` | DOM 렌더링 및 이벤트 핸들링 |
| `App` | 초기화 및 상태 관리 진입점 |

| 항목 | 기준 |
|------|------|
