# AI Dream Journal - 제품 요구사항 정의서 (PRD)

## 1. 프로젝트 개요

### 1.1 목적
AI 기반 꿈 일기 앱. 꿈 기록, AI 꿈 해석, 상징 분석, 반복 패턴 발견, 주간 꿈 인사이트 제공.

### 1.2 기술 스택
- 프레임워크: Next.js 14 (App Router)
- 언어: TypeScript
- 데이터베이스: PostgreSQL + Drizzle ORM
- UI: shadcn/ui + Tailwind CSS
- 차트: Recharts
- AI: AI SDK + Open Router
- 모델: `anthropic/claude-haiku-4.5`
- 컨테이너: Docker (PostgreSQL - 기존 사용)

### 1.3 대상 사용자
- 꿈을 기록하고 싶은 사람
- 꿈의 의미를 알고 싶은 사람
- 꿈 패턴을 분석하고 싶은 사람
- 꿈 일기를 체계적으로 관리하고 싶은 사람

## 2. 기능 (9개)

### 기능 1: 꿈 일기 CRUD
꿈 기록 관리
- 꿈 작성/수정/삭제
- 제목, 내용, 날짜
- 감정 기록 (긍정/부정/중립)
- 생생함 정도 (1-5)
- 태그 부여

### 기능 2: 태그 시스템
태그 관리
- 태그 생성/삭제
- 꿈에 여러 태그 부여
- 태그별 통계
- 자주 나오는 요소 (인물, 장소, 물건)

### 기능 3: 캘린더 뷰
날짜별 꿈 보기
- 월별 캘린더
- 꿈 있는 날 표시
- 날짜 선택하여 꿈 보기
- 감정에 따라 색상 구분

### 기능 4: 검색 & 필터
검색 및 필터
- 전체 검색 (제목, 내용)
- 날짜 범위
- 태그별
- 감정별
- 생생함별

### 기능 5: 통계 대시보드
꿈 통계
- 월별 기록 일수
- 감정 분포 (Pie Chart)
- 생생함 추이 (Line Chart)
- 자주 나오는 태그 (Bar Chart)
- 꿈 기록 스트릭

### 기능 6: AI 꿈 해석 ⭐
AI 꿈 해석
- 꿈 내용 입력 → AI 해석
- 심리학적 관점
- 상징적 의미
- 가능한 메시지
- 해석 저장

### 기능 7: AI 상징 분석 ⭐
AI 상징 분석
- 꿈 속 요소 자동 추출
- 각 상징의 의미
- 개인화된 해석
- 상징 데이터베이스 구축

### 기능 8: AI 패턴 발견 ⭐
AI 패턴 탐지
- 반복되는 꿈 테마
- 자주 나오는 인물/장소
- 감정 패턴
- 시간대별 패턴
- 패턴 의미 해석

### 기능 9: AI 주간 인사이트 ⭐
AI 주간 분석
- 한 주간 꿈 요약
- 주요 테마
- 감정 흐름
- 잠재의식 메시지
- 다음 주 관찰 포인트

## 3. 데이터 구조

### 3.1 테이블 (4개)

#### dreams (꿈 기록)
```typescript
{
  id: string (uuid)
  title: string
  content: text
  date: date
  emotion: enum ('positive', 'neutral', 'negative')
  vividness: number (1-5)
  lucid: boolean (자각몽 여부)
  createdAt: timestamp
  updatedAt: timestamp
}
```

#### interpretations (AI 해석)
```typescript
{
  id: string (uuid)
  dreamId: string (FK → dreams)
  interpretation: text (전체 해석)
  psychological: text (심리학적 관점)
  symbolic: text (상징적 의미)
  message: text (메시지)
  analyzedAt: timestamp
}
```

#### symbols (상징)
```typescript
{
  id: string (uuid)
  dreamId: string (FK → dreams)
  symbol: string (상징 이름)
  category: enum ('person', 'place', 'object', 'action', 'emotion')
  meaning: text (의미)
  frequency: number (등장 횟수 - 전체 꿈에서)
  createdAt: timestamp
}
```

#### patterns (패턴)
```typescript
{
  id: string (uuid)
  type: enum ('theme', 'person', 'place', 'emotion')
  name: string (패턴 이름)
  description: text (패턴 설명)
  occurrences: number (발생 횟수)
  dreamIds: text[] (관련 꿈 ID 배열)
  significance: text (의미)
  createdAt: timestamp
  updatedAt: timestamp
}
```

## 4. AI 구현

### 4.1 AI SDK 설정
```bash
npm install ai @ai-sdk/openai
```

```typescript
import { createOpenAI } from '@ai-sdk/openai'

const openrouter = createOpenAI({
  baseURL: 'https://openrouter.ai/api/v1',
  apiKey: process.env.OPENROUTER_API_KEY,
})

const model = openrouter('anthropic/claude-haiku-4.5')
```

### 4.3 AI 함수

#### interpretDream(content: string)
```typescript
입력: {
  title: "하늘을 나는 꿈",
  content: "나는 새처럼 하늘을 자유롭게 날고 있었다. 구름 위를 날면서 매우 행복했다.",
  emotion: "positive",
  vividness: 5
}

출력: {
  interpretation: "자유와 해방을 갈망하는 마음의 표현입니다...",
  psychological: "현재 제약이나 스트레스에서 벗어나고 싶은 욕구를 나타냅니다...",
  symbolic: "비행은 한계를 초월하는 상징으로, 개인적 성장을 의미합니다...",
  message: "당신은 새로운 가능성을 탐색할 준비가 되어 있습니다..."
}
```

#### extractSymbols(content: string)
```typescript
입력: "어두운 숲 속에서 늑대를 만났다. 처음엔 무서웠지만 늑대가 나를 안내했다."

출력: [
  {
    symbol: "숲",
    category: "place",
    meaning: "무의식의 세계, 미지의 영역"
  },
  {
    symbol: "늑대",
    category: "person",
    meaning: "본능, 내면의 힘, 안내자"
  },
  {
    symbol: "어둠",
    category: "emotion",
    meaning: "두려움, 불확실성"
  }
]
```

#### detectPatterns(dreams: Dream[])
```typescript
입력: [
  { content: "물 속에 빠졌다...", date: "2026-01-10" },
  { content: "바다에서 수영했다...", date: "2026-01-15" },
  { content: "비가 많이 왔다...", date: "2026-01-20" }
]

출력: [
  {
    type: "theme",
    name: "물 관련 꿈",
    description: "물이 반복적으로 등장하는 패턴",
    occurrences: 3,
    significance: "감정의 흐름과 무의식을 상징. 정화나 변화의 시기를 나타낼 수 있음"
  }
]
```

#### generateWeeklyInsight(dreams: Dream[])
```typescript
입력: 이번 주 꿈 목록 (5개)

출력: {
  summary: "이번 주는 변화와 성장을 주제로 한 꿈이 많았습니다...",
  mainThemes: ["자유", "여행", "새로운 시작"],
  emotionalFlow: "긍정적 감정이 70%로 우세했으며, 전반적으로 희망적인 기운이 느껴집니다.",
  subconscious: "새로운 도전에 대한 준비가 되어 있으며, 변화를 두려워하지 않는 마음 상태입니다.",
  nextWeek: "다음 주는 대인 관계나 협력과 관련된 꿈이 나타날 수 있습니다."
}
```

### 4.4 AI 프롬프트 템플릿

#### 꿈 해석 프롬프트
```
당신은 꿈 해석 전문가이자 심리학자입니다.

다음 꿈을 분석해주세요:

[꿈 정보]
제목: {title}
내용: {content}
감정: {emotion}
생생함: {vividness}/5

다음 관점에서 해석해주세요:
1. 전체적인 해석
2. 심리학적 관점 (프로이트, 융 등)
3. 상징적 의미
4. 이 꿈이 전하는 메시지

다음 JSON 형식으로 응답:
{
  "interpretation": "전체 해석",
  "psychological": "심리학적 관점",
  "symbolic": "상징적 의미",
  "message": "메시지"
}
```

#### 상징 추출 프롬프트
```
당신은 꿈 상징 분석 전문가입니다.

다음 꿈에서 의미 있는 상징들을 추출해주세요:

[꿈 내용]
{content}

상징 추출 기준:
- 인물 (person): 사람, 동물
- 장소 (place): 위치, 환경
- 물건 (object): 사물
- 행동 (action): 주요 행위
- 감정 (emotion): 강한 감정

각 상징의 일반적 의미를 설명해주세요.

JSON 배열 형식으로 응답:
[
  {
    "symbol": "상징 이름",
    "category": "person|place|object|action|emotion",
    "meaning": "의미 설명"
  }
]
```

#### 패턴 탐지 프롬프트
```
당신은 꿈 패턴 분석 전문가입니다.

다음은 최근 꿈들의 목록입니다:

[꿈 목록]
{dreams}

반복되는 패턴을 찾아주세요:
1. 테마 (주제)
2. 인물 (자주 나오는 사람/동물)
3. 장소 (반복되는 장소)
4. 감정 (일관된 감정)

각 패턴의 의미를 해석해주세요.

JSON 배열 형식:
[
  {
    "type": "theme|person|place|emotion",
    "name": "패턴 이름",
    "description": "패턴 설명",
    "occurrences": 횟수,
    "significance": "의미"
  }
]
```

#### 주간 인사이트 프롬프트
```
당신은 꿈 분석 전문가입니다.

이번 주 꿈들을 분석해주세요:

[이번 주 꿈들]
{dreams}

다음 내용을 포함해서 분석:
1. 한 주 요약
2. 주요 테마 (3가지)
3. 감정 흐름 분석
4. 잠재의식이 전하는 메시지
5. 다음 주 관찰 포인트

친근하고 통찰력 있는 톤으로 작성해주세요.

JSON 형식:
{
  "summary": "요약",
  "mainThemes": ["테마1", "테마2", "테마3"],
  "emotionalFlow": "감정 흐름",
  "subconscious": "잠재의식 메시지",
  "nextWeek": "다음 주 포인트"
}
```
